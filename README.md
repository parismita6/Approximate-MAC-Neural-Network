# Approximate MAC Unit Optimization using Neural Networks

## 📌 Overview
This project focuses on improving Multiply-Accumulate (MAC) unit performance using approximate multiplier designs. These optimizations are applied to image processing tasks and evaluated on neural networks.

## ⚙️ Key Features
- Approximate 8x8 multiplier implementations
- Multiple design variations (MULT_1, MULT_2, MULT_3, MULT_4)
- Integration with PyTorch
- Performance vs accuracy trade-off analysis

## 🧠 Models Used
- MNIST dataset (handwritten digit classification)
- ResNet architecture
